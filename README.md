# todo-applet

Inspired by rexims todo-rs

`but in C!`

## usage
| key | description |
| --- | --- |
| <kbd>j</kbd> | move downwards |
| <kbd>k</kbd> | move upwards |
| <kbd>Enter</kbd> | toggle done-status |
| <kbd>a</kbd> | add item |
| <kbd>d</kbd> | remove selected item |
| <kbd>i</kbd> | enter insert mode, exit with <kbd>Enter</kbd> |
